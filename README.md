## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- get-domain-url
- router
- express-timestamp


## DevDependencies

- Mocha/Chai
- Nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
